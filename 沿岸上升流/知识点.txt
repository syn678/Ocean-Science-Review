# 沿岸上升流相关知识点整理

## 一、基础理论（理解上升流的前提）
1. 位涡守恒
大洋环流遵循位涡守恒，核心关系：
Q = (f + ζ) / H = 常数
- f：行星涡度（科氏参数，纬度越高绝对值越大）
- ζ：相对涡度（水体自身旋转）
- H：水柱厚度

三种平衡情况：
1）同纬度，H变化 → 相对涡度变化
2）H不变，南北移动 → f变化，相对涡度调整
3）ζ≈0时，f与H同步变化，形成Sverdrup平衡基础

2. Ekman输运
风作用下，表层海水净输运方向与风向垂直：
- 北半球：风向右侧90°
- 南半球：风向左侧90°

3. Ekman抽吸
Ekman输运辐合或辐散，产生垂直运动：
- 辐散 → 海水上升（上升流）
- 辐合 → 海水下沉（下降流）

4. Sverdrup平衡
风引起的Ekman抽吸使水柱拉伸或压缩，为保持位涡守恒，海水会发生南北向运动，形成大尺度经向输运，是副热带环流的主要动力。

5. 西边界流
Sverdrup输运在大洋西侧汇聚，形成强而窄的西边界流，如湾流。西边界流多伴随下降流，而大洋东侧多出现上升流。

## 二、沿岸上升流核心内容
1. 形成机制
沿岸风平行于海岸吹，使Ekman输运方向**离岸**，表层水被吹走，下层冷水向上补充，形成沿岸上升流。

2. 南北半球触发条件
- 北半球：风平行海岸向南吹 → 海水向右离岸 → 上升流
- 南半球：风平行海岸向北吹 → 海水向左离岸 → 上升流

反之，Ekman输运向岸则形成沿岸下降流。

3. 主要特征
- 表层水温偏低
- 温跃层抬升，靠近海面
- 底层营养盐上涌，生产力极高
- 常伴随多雾、凉爽的沿岸气候

4. 与大洋上升流的区别
- 沿岸上升流：局地小尺度、强度大、生态效应极强
- 大洋上升流：大尺度、强度弱、影响整体环流与生产力

5. 位涡守恒在沿岸上升流中的体现
上升流使水柱拉伸（H增大），在纬度基本不变时，相对涡度增加，易形成沿岸气旋式环流。

---

## 三、典型沿岸上升流区域
1. 秘鲁—智利沿岸（洪堡寒流）
世界最强上升流区，形成著名秘鲁渔场，与ENSO密切相关。

2. 美国加利福尼亚沿岸（加利福尼亚寒流）
典型东边界上升流，影响沿岸气候与渔业。

3. 非洲西南部沿岸（本格拉寒流）
强上升流区，渔业资源丰富，易出现底层缺氧现象。

4. 北非西海岸（加那利寒流）
北半球重要上升流区，支持西北非渔业生产。

---

## 四、整体逻辑链
1. 位涡守恒是大洋运动的基本约束
2. 风驱动Ekman输运，产生辐合辐散
3. 大尺度形成Sverdrup平衡与副热带环流
4. 西边界形成强西边界流
5. 沿岸风导致离岸Ekman输运 → 沿岸上升流
6. 上升流带来营养盐 → 高生产力 → 世界级渔场
